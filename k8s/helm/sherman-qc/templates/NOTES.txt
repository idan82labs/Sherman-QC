1. CREATE AI API KEY SECRET
   The application requires an AI provider API key. Create the secret:

   # For Claude (recommended)
   kubectl create secret generic {{ .Values.ai.anthropicApiKeySecret.name }} \
     --from-literal={{ .Values.ai.anthropicApiKeySecret.key }}=YOUR_ANTHROPIC_API_KEY

   # For Gemini
   kubectl create secret generic {{ .Values.ai.googleApiKeySecret.name }} \
     --from-literal={{ .Values.ai.googleApiKeySecret.key }}=YOUR_GOOGLE_API_KEY

   # For OpenAI
   kubectl create secret generic {{ .Values.ai.openaiApiKeySecret.name }} \
     --from-literal={{ .Values.ai.openaiApiKeySecret.key }}=YOUR_OPENAI_API_KEY
